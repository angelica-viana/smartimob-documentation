# Requisitos Não Funcionais

## RNF01 — Aplicação Web

O sistema deverá ser acessível via navegador.

---

## RNF02 — Estrutura Escalável

A arquitetura deverá permitir expansão futura.

---

## RNF03 — Banco de Dados Relacional

O banco deverá possuir modelagem organizada e relacional.

---

## RNF04 — Interface Intuitiva

A interface deverá ser simples e objetiva.

---

## RNF05 — Navegação Organizada

As funcionalidades deverão possuir acesso intuitivo.

---

## RNF06 — Performance

O sistema deverá possuir tempo de resposta adequado.

---

## RNF07 — Segurança

Os dados deverão ser protegidos contra acesso não autorizado.

---

## RNF08 — Backup

O sistema deverá possuir rotina de backup.

---

## RNF09 — Compatibilidade

Compatível com:

- Google Chrome;
- Edge;
- Firefox.

---

## RNF10 — Responsividade

Compatível com:

- Desktop;
- Tablet;
- Smartphone.

---

## RNF11 — Escalabilidade

O sistema deverá suportar:

- Novos módulos;
- APIs;
- Integrações;
- Crescimento de usuários.

---

## RNF12 — Exportação PDF

Os recibos deverão possuir exportação PDF.
