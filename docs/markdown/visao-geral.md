# Visão Geral do Projeto

## Nome do Sistema
SmartImob

---

## Objetivo

Desenvolver uma aplicação web para auxiliar na gestão imobiliária, com foco em:

- Controle de contratos de locação
- Geração mensal de recibos
- Controle de reajustes
- Controle de vencimentos
- Gestão financeira básica
- Organização de proprietários e inquilinos

---

## Objetivo Futuro

A aplicação deverá ser construída inicialmente de forma simples, porém preparada para crescimento futuro, permitindo novas funcionalidades sem necessidade de reconstrução do sistema.

---

## Perfis de Usuário

### Administrador
Responsável pela gestão completa do sistema.

Permissões:
- Cadastro de imóveis
- Cadastro de contratos
- Emissão de recibos
- Controle financeiro
- Gestão de reajustes
- Visualização de alertas

---

### Usuário Operacional (futuro)

Perfil com acesso limitado apenas à emissão e consulta de recibos.
