# Estrutura Visual das Telas

## Estrutura Geral

O sistema deverá possuir:

- Cabeçalho superior;
- Menu lateral fixo;
- Área central de conteúdo;
- Alertas visuais;
- Navegação simplificada.

---

## Tela de Login

Campos:

- E-mail;
- Senha.

Botões:

- Entrar;
- Recuperar senha.

---

## Dashboard

Componentes:

- Cards resumo;
- Alertas importantes;
- Ações rápidas;
- Indicadores financeiros.

---

## Tela de Proprietários

Componentes:

- Busca;
- Tabela;
- Botão novo cadastro;
- Editar;
- Excluir.

---

## Tela de Contratos

Componentes:

- Filtros;
- Listagem;
- Controle de status;
- Reajustes;
- Emissão de recibos.

---

## Tela de Recibos

Componentes:

- Emissão;
- Prévia;
- Exportação PDF;
- Histórico;
- Reimpressão.

---

## Responsividade

Todas as telas deverão funcionar em:

- Computadores;
- Tablets;
- Smartphones.
