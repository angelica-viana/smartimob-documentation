# Requisitos Funcionais

## RF01 — Autenticação e Controle de Acesso

O sistema deverá:

- Permitir login com e-mail e senha;
- Permitir recuperação de senha;
- Manter sessão autenticada;
- Permitir logout;
- Controlar permissões por perfil.

---

## RF02 — Dashboard Inicial

O dashboard deverá apresentar:

- Contratos ativos;
- Contratos próximos do vencimento;
- Reajustes previstos;
- Recibos pendentes;
- Total recebido no mês;
- Alertas importantes;
- Resumo financeiro.

---

## RF03 — Cadastro de Proprietários

O sistema deverá permitir:

- Cadastrar proprietários;
- Editar informações;
- Excluir registros;
- Consultar proprietários.

Campos obrigatórios:

- Nome;
- CPF/CNPJ;
- Telefone;
- E-mail.

---

## RF04 — Cadastro de Inquilinos

O sistema deverá permitir:

- Cadastro;
- Edição;
- Exclusão;
- Consulta de inquilinos.

---

## RF05 — Cadastro de Imóveis

O sistema deverá permitir:

- Cadastro de imóveis;
- Edição;
- Exclusão;
- Consulta;
- Controle de status.

Status:

- Vago;
- Ocupado;
- Suspenso.

---

## RF06 — Gestão de Contratos

O sistema deverá permitir:

- Criar contratos;
- Editar contratos;
- Encerrar contratos;
- Renovar contratos;
- Gerenciar vencimentos;
- Gerenciar reajustes.

---

## RF07 — Gestão Financeira

O sistema deverá permitir:

- Lançamento de IPTU;
- Condomínio;
- Água;
- Luz;
- Outros débitos;
- Descontos.

O sistema deverá calcular:

- Valor bruto;
- Valor líquido.

---

## RF08 — Gestão de Reajustes

O sistema deverá:

- Identificar contratos próximos do reajuste;
- Aplicar reajuste automático;
- Permitir reajuste manual;
- Registrar histórico.

Índices suportados:

- IGP-M;
- Manual.

---

## RF09 — Gestão de Recibos

O sistema deverá:

- Gerar recibos mensais;
- Gerar PDF;
- Permitir reimpressão;
- Registrar histórico.

---

## RF10 — Sistema de Alertas

O sistema deverá emitir alertas para:

- Contratos vencendo;
- Contratos vencidos;
- Reajustes próximos;
- Recibos pendentes.

---

## RF11 — Busca e Filtros

O sistema deverá permitir buscas por:

- Proprietário;
- Inquilino;
- Endereço;
- CPF.

Filtros:

- Contratos ativos;
- Contratos vencidos;
- Reajustes do mês.

---

## RF12 — Responsividade

O sistema deverá funcionar em:

- Desktop;
- Tablet;
- Smartphone.
